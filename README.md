# REAL-TSE 2026 - Team ChuEst

Code, configs and outputs for our REAL-TSE 2026 challenge submissions.

- track1: causal block-online USEF-TFGridNet, submission #279 (Online track)
- track2: USEF-TFGridNet extractor + RE-USE enhancer with per-file routing, submission #188 (Offline track)
- paper: system description (PDF)

Model weights are attached to the release because of file size limits.
The RE-USE enhancer weights are not redistributed here (research and development
only license). Get them from https://huggingface.co/nvidia/RE-USE
